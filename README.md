# Triathlon Training Coach

A React + TypeScript application for generating personalized triathlon training plans.

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run the development server:
   ```bash
   npm run dev
   ```

3. Build for production:
   ```bash
   npm run build
   ```

## Features
- Goal-based training plans (Sprint, Olympic, 70.3, Full)
- Interactive landing page
- Personalized schedule generation
